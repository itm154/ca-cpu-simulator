# CPU Simulator built in Rust

## Usage

Prerequisites:

- Rust and `cargo` [(Install)](https://www.rust-lang.org/learn/get-started)

Running:

```bash
$ cargo run --quiet
```

### Instruction Sets

- HALT
- LVAL
- LOAD
- STORE
- ADD
- SUB
- JMP
- MOV
